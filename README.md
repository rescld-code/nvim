Neovim Configuration

## Require Env

- neovim >= v0.5.0
- python
	- pip
	- pynvim
- node.js >= v8.0
	- npm
	- yarn
- fzf

## Language Server

- C/C++
	- clangd
- Python
	- python-language-server
- Golang
	- vim-go

## Install

```shell
git clone https://github.com/rescld-code/nvim.git ~/.config/nvim
```
